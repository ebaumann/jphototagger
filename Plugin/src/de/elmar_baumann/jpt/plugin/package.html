Classes supporting Plugins for Photo Tagger.

How to write a Plugin

  • Include the JAR of this package into Your plugin project
  • Extend de.elmar_baumann.jpt.plugin.Plugin
  • Create a JAR with that class, a META-INF/services directory and a file de.elmar_baumann.jpt.plugin.Plugin in that directory
  • That file must contain at least on line with the fully qualified name of Your class that extends de.elmar_baumann.jpt.plugin.Plugin
  • Of course You can include many classes into the JAR used by the class extending de.elmar_baumann.jpt.plugin.Plugin
  • Contact the distributor to integrate Your plugin into the soure code's version control
