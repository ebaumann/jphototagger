Einstellungen, Kartenreiter "Vorschaubilder"

Maximale Seitenlänge berechneter Vorschaubilder in Pixel

Die Vorschaubilder werden so groß berechnet wie Sie hier angeben. Ändern Sie die Größe (später), sollten Sie die Vorschaubilder neu berechnen lassen. Klicken Sie dazu auf die Schaltfläche Alle neu berechnen. Während der Aktualisierung können Sie den Einstellungen-Dialog schließen und weiterarbeiten. Der Fortschritt wird in einem anderen Dialog angezeigt, den Sie nicht schließen sollten, außer Sie wollen keine weiteren Vorschaubilder berechnen lassen.

Wie sollen die Vorschaubilder erzeugt werden?

Sie können sofort sehen, wie sich eine Einstellung auswirkt: Schieben Sie den Einstellungen-Dialog zur Seite, wählen ein Vorschaubild aus, klicken mit der linken Maustaste darauf und wählen aus dem Kontextmenü Aktualisieren > Vorschaubild. Falls Ihnen das Ergebnis nicht gefällt, probieren Sie eine andere Option und lassen erneut ein Vorschaubild erzeugen.

Erzeuge Vorschaubilder mit Java Image IO

Die Vorschaubilder werden mit der Java Image IO (http://java.sun.com/javase/6/docs/technotes/guides/imageio/index.html) erzeugt. Von RAW-Dateien und 16 Bit-TIFF-Dateien können so keine Vorschaubilder erzeugt werden.

Wählen Sie diese Option, falls Sie ausschließlich JPEG-Dateien haben und/oder nur 8 Bit-TIFF-Dateien.

Diese Vorgabe ist der Standard.

Erzeuge Vorschaubilder mit Imagero

Die Vorschaubilder werden mit Imagero (http://reader.imagero.com/) erzeugt. Von RAW-Dateien können so keine Vorschaubilder erzeugt werden.

Wählen Sie diese Option, falls Sie außer JPEG-Dateien 16 Bit-TIFF-Dateien haben.

Benutze in den Bildern enthaltene Vorschaubilder (erzeuge keine)

Kreuzen Sie diese Option an, werden die Vorschaubilder geladen, die in den Bildern eingebettet sind. Das geht schneller, als die Vorschaubilder zu berechnen, aber manchmal sind die eingebetteten Vorschaubilder von geringer Qualität.

Wählen Sie diese Option, falls Sie RAW-Dateien haben und JPhotoTagger die Vorschaubilder auch extrahieren kann (falls das nicht der Fall ist, erkennen Sie das sofort) und kein eigenes Programm zum Erzeugen einstellen wollen, siehe unten. JPhotoTagger kann nur bei einigen RAW-Dateien die Vorschaubilder auslesen, bei vielen wird das nicht funktionieren.

Erzeuge Vorschaubilder mit eigenem Programm

Eigenes Programm zum Erzeugen der Vorschaubilder

Sie können Vorschaubilder mit einem beliebigen anderen Programm erzeugen lassen, sofern dieses JPEGs in die Standardausgabe schreiben kann. Die Schaltfläche Auswählen bietet vorgefertigte Skripte an.

Wählen Sie diese Option, falls:

  • Sie bei RAW-Dateien nicht die eingebetteten Vorschaubilder haben wollen
  • Ihr Programm hochwertige Vorschaubilder liefert
  • Ihr Programm Vorschaubilder schneller liefert
