Glossar: XMP-Filialdatei

Eine XMP-Filialdatei ist eine Textdatei im XMP-Format im gleichen Ordner wie die Bilddatei und hat den gleichen Namen, nur die Endung lautet xmp: bild-01.xmp ist die Filialdatei von bild-01.nef (von bild-01.jpg, von bild-01.tiff, ...). Sie enthält Metadaten über das zugehörige Bild.
