JPhotoTagger-Daten exportieren

Über das Menü Datei > Exportieren > Verschiedene JPhotoTagger-Daten können Sie alle Daten exportieren, die JPhotoTagger (wieder) importieren kann.

In den Ordner, den Sie auswählten, werden Dateien mit standardisierten Namen geschrieben. Existieren bereits Dateien dieses Namens, werden diese ohne Rückfrage überschrieben! Wollen Sie mit dem Export Daten sichern und mehrere Datei-Versionen behalten, exportieren Sie die Daten in unterschiedliche Ordner, beispielsweise mit dem Datum als Namen, z.B. 2010-03-10 für den 10.03.2010.

Nutzen Sie diese Funktion zum Sichern und/oder zum Austauschen zwischen verschiedenen JPhotoTagger-Datenbanken.
