Kontextmenü im Vorschaubildfenster: Aktualisieren > Vorschaubild aktualisieren

Dieser Befehl aktualisiert im den Datenbankordner untergeordneten Ordner thumbnails die Vorschaubilder der ausgewählten Bilder.

Sinnvoll ist das:

  • zum Testen eines eigenen Programms, das Vorschaubilder erzeugt siehe Einstellungen-Dialog, Kartenreiter (Vorschaubilder)
  • falls Sie testen wollen, ob die Java Image IO Ihnen gefälligere Bilder liefert oder Imagero
  • nachdem Sie die maximale Seitenlänge eines Vorschaubilds änderten und anschließend nicht im Einstellungen-Dialog, Kartenreiter Vorschaubilder, alle Vorschaubilder neu berechnen ließen
  • falls die Vorschaubilder noch nicht berechnet wurden und Sie nicht warten wollen, bis das automatisch geschieht
