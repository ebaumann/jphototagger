Einstellungen, Kartenreiter "Plugins"

In diesem Dialog zeigen Plugins ihre eigenen Hilfeseiten ein, sofern diese darüber verfügen. Klicken Sie hierzu auf die Schaltfläche Hilfe (F1 zeigt nur diesen Text an). Im PDF-Handbuch von JPhotoTagger steht deren Inhalt nicht.

Plugins sind Java-Bibliotheken, die die sogenannte "Plugin-Schnittstelle" von JPhotoTagger implementieren und so dessen Funktionalität erweitern. Sie sehen die Plugins im Kontextmenü des Vorschaubildfensters, Menü-Eintrag Plugins.
