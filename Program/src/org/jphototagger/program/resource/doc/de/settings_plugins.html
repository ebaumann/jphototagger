Einstellungen, Kartenreiter "Plugins"

In diesem Hilfe-Dialog fügen Plugins ihre eigenen Hilfeseiten ein, sofern diese darüber verfügen. Im PDF-Handbuch von JPhotoTagger steht deren Inhalt nicht.

Plugins sind Java-Bibliotheken, die die sogenannte "Plugin-Schnittstelle" von JPhotoTagger implementieren und so dessen Funktionalität erweitern. Sie sehen die Plugins im Kontextmenü des Vorschaubildfensters, Menüeintrag Plugins. Ein Plugin verarbeitet ausgewählte Dateien.
