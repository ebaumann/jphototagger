Plugins: Einführung

Plugins sind zusätzliche Fähigkeiten, die JPhotoTagger bereitstellt für ausgewählte Dateien im Vorschaubildfenster. Sie können ein Plugin nutzen über den Kontextmenü-Eintrag Plugins.

Falls Sie ein Plugin nicht benutzen wollen, können Sie es im Einstellungen-Dialog deaktivieren im Kartenreiter Plugins.
