Einstellungen, Kartenreiter "Sonstiges"

Kartenreiter "Externe Programme"

Hier wählen Sie Programme aus, die benutzt werden zum Anzeigen unterschiedlichen Inhalts.

Webbrowser

Der Webbrowser wird aufgerufen, falls Sie mit dem Internet verbunden werden müssen, beispielsweise zum Anzeigen von Satellitenbildern des Aufnahmeorts bei GPS-Daten im Bild.

PDF-Betrachter

Der PDF-Betrachter wird benutzt zum Anzeigen des PDF-Handbuchs.

Kartenreiter "Datenbankordner"

Sie können die Datenbank in einem anderen Ordner anlegen lassen. Wollen Sie Ihre bisherigen Daten übernehmen, kopieren Sie aus dem aktuellen Datenbankordner, der im Dialog angezeigt wird, folgende Dateien nach Beenden von JPhotoTagger und vor einem Neustart:

  • Den Ordner thumbnails, dieser enthält die Vorschaubilder
  • Alle Dateien, die mit database starten, z.B. database.data

So können z.B. mehrere - nicht gleichzeitig! - die selbe Datenbank benutzen und bearbeiten oder Sie wählen einen Ordner aus, der täglich automatisch gesichert wird, sodass das Backup der Datenbank gewährleistet ist.

Kartenreiter "Protokoll"

Was / wie viel soll protokolliert werden?

Der Programmablauf wird protokolliert und Programmfehler in eine Logdatei geschrieben. Diese kann angezeigt werden über das Menü Fenster > Fehlerprotokoll. Bedeutungen:

  • WARNUNG: Nur wichtige Fehler und weniger wichtige werden protokolliert. Warnungen werden bei anderen Einstellungen zusätzlich protokolliert außer bei SCHWERWIEGEND.
  • SCHWERWIEGEND: Nur wichtige Fehler werden protokolliert. Schwer wiegendes wird bei allen anderen Einstellungen zusätzlich protokolliert.
  • INFO: Informative Ausgaben, beispielsweise welche Verzeichnisse gerade automatisch eingelesen werden. Dies ist die Grunddeinstellung.
  • KONFIG: Ausgaben betreffend die Konfiguration, für den Benutzer eines Programms meist unwichtig.
  • FEIN: Detaillierte Ausgaben, insbesondere SQL-Statements, die die Datenbank aktualisieren
  • FEINER: Noch detailliertere Ausgaben, auch SQL-Statements, die die Datenbank abfragen und nicht modifizieren
  • AM FEINSTEN: Detailliertere Informationen über den Programmablauf

Wählen Sie WARNUNG oder INFO, solange keine Probleme bestehen und zur Fehlerbehebung AM FEINSTEN. Meldungen anderer Kategorien als WARNUNG oder SCHWERWIEGEND werden in die Standardausgabe des Betriebssystems geschrieben, andernfalls würde die Protokolldatei viele Megabyte groß. Die Standardausgabe sehen Sie im Fenster Systemausgabe, zu erreichen über die Schaltfläche Ausgabe am unteren Fensterrand.

Format

  • Strukturiert: Nachrichten können sortiert werden nach ihrer Kategorie, jede Nachricht kann einzeln ausgewählt werden und betrachtet im Detail und Sie können filtern und suchen innerhalb der Nachrichten. Dies ist die Grundeinstellung.
  • Textzeilen: Die Nachrichten werden Zeile für Zeile untereinander angezeigt. Sie können die Zeilen betrachten und haben sonst keine weitere Optionen. Bei dieser Einstellung benötigt das Protokollieren am wenigsten Speicherplatz.

Kartenreiter "Verschiedenes"

Versteckte Ordner einlesen und anzeigen

Wollen Sie Bilder archivieren, die in versteckten Ordnern gespeichert sind, kreuzen Sie dieses Kästchen an.

Nach dem Programmstart automatisch auf eine neue Version prüfen

Ist diese Option angekreuzt, sucht JPhotoTagger im Internet nach einer neueren Programmversion. Existiert eine, wird diese zum Download angeboten.

Kopieren/Verschieben von Dateien: Quelldatei

Nachfragen, falls gleichnamige Datei im Zielordner existiert

Ist diese Option aktiv, wird erst nachgefragt, ob eine existierende Zieldatei überschrieben werden soll, bevor die Datei kopiert wird. Beim Verschieben hat diese Option keine Auswirkung.

Automatisch umbenennen , falls gleichnamige Datei im Zielordner existiert

Ist diese Option angekreuzt, wird die Quelldatei automatisch umbenannt, falls im Zielordner eine gleichnamige Datei existiert: Vor der Endung wird ein Unterstrich angehängt gefolgt von einer fortlaufenden Nummer.
