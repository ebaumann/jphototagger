Verschiedene JPhotoTagger-Daten importieren

Über das Menü Datei > Importieren > Verschiedene JPhotoTagger-Daten können Sie Daten importieren, die JPhotoTagger exportiert hat.

Im ausgewählten Ordner müssen die exportierten Dateien existieren, und die gleichen Namen haben, unter die sie JPhotoTagger beim Export speicherte.
