Stichwörter exportieren

Über das Menü Datei > Exportieren > Stichwortbaum öffnen Sie den Stichwortbaum-Exportdialog. Mit diesem können Sie Stichwörter exportieren für JPhotoTagger oder ein anderes Programm.

Die Stichwörter werden in verschiedene Formate exportiert. Es gilt das gleiche wie unter Stichwörter importieren beschrieben.
