Bilder importieren

Sie können Bilder von der Kamera-Speicherkarte oder einem anderen Ordner einschließlich dessen untergeordneten Ordnern in einen (anderen) Ordner kopieren lassen.

Rufen Sie den Import-Dialog auf über das Menü Datei > Bilder importieren, Tastenkürzel Strg+Umschalt+P, wählen zwei unterschiedliche Ordner aus und klicken auf Ok.

JPhotoTagger sucht im ausgewählten Ordner einschließlich dessen untergeordneten Ordnern nach Bildern und kopiert alle gefundenen in den Zielordner.

Existiert eine Datei gleichen Namens im Zielordner, wird der kopierten Datei eine Nummer hinzugefügt, Dateien werden nicht überschrieben.

Anschließend werden alle importierten Bilder im Album Zuletzt importiert angezeigt.
