Dialog: Zuletzt eingegebene Begriffe

Dieser Dialog zeigt Ihnen an, welche Begriffe Sie zuletzt eingaben in die Felder des Kartenreiters Bearbeiten.

Sie können den Dialog anzeigen mit der Funktionstaste F4. Er ist so lange zu sehen, bis Sie ihn wieder schließen.

Mit den Tasten ALT + TAB (Tabulator) wechseln Sie während des Bearbeitens zu diesen Dialog. Drücken Sie Alt + A oder Alt + B usw. - den Buchstaben in den eckigen Klammern - wird der Begriff eingefügt. Der zuletzt eingegebene Begriff ist immer dem Buchstaben A zugeordnet, der vorletzte B usw. Mit ALT + TAB gelangen Sie zurück zum Bearbeiten-Kartenreiter.
