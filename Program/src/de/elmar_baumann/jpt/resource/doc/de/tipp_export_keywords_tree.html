Exportieren Sie regelmäßig Ihren Stichwortbaum

Exportieren Sie regelmäßig Ihren Stichwortbaum über das Menü Datei > Exportieren > Stichwortbaum, Format JPhotoTagger, denn dieser wird nicht in die XMP-Filialdateien geschrieben (Im Lightroom-Format werden Hilfsstichwörter nicht exportiert, deshalb eignet sich das Format nicht für einen späteren Import in JPhotoTagger!).

So können Sie den Stichwortbaum auf andere Rechner oder andere JPhotoTagger-Datenbanken übertragen über den Menüpunkt Datei > Importieren > Stichwortbaum, und in eine neue Datenbank einlesen, falls die alte gelöscht oder beschädigt wurde.
