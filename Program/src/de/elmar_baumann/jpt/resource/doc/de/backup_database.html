Datenbank sichern

Der Menüeintrag Datei > Datenbank sichern lässt Sie einen Ordner auswählen, in den JPhotoTagger nach Schließen der Datenbank und vor Beenden der Sitzung die Datenbankdatei speichert.

Dem Namen wird das aktuelle Datum angehängt. Existiert dort bereits eine gleichnamige Datei, wird der Name abgeändert, sodass keine Datei überschrieben wird.
