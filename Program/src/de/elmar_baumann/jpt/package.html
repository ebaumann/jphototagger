Some Implementation notes on JPhotoTagger.

The main parts are:

  • A database for specific persistent content. A database notifies database listeners on database changes.
  • Models containing specific database content. A database model listens to changes within its database and updates in that case the model content.
  • Views displaying different models; some models have a specific renderer.
  • Controllers are listening to user actions and modifying the database or doing something else. A controller uses other classes to perform something so that the other classes can be re-used.
