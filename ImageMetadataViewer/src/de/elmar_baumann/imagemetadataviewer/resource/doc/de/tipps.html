Tipps

  • Speichern Sie wichtige Suchen! So finden Sie schnell Bilder mit bestimmten Eigenschaften. Eine gespeicherte Suche ist eine Vorlage für eine ähnliche neue.
  • Haben Sie Metadaten eines Bilds verändert, das bereits in der Datenbank ist, das Dateidatum aber auf das alte zurückgesetzt, wählen Sie das Bild aus, klicken die rechte Maustaste und wählen Text-Metadaten aktualisieren, keine Thumbnails (schneller). Sie können ganze Verzeichnisse auf diese Weise aktualisieren über den Verzeichnisse scannen-Dialog.
  • Überprüfen Sie gelegentlich Ihre Einstellungen (Bearbeiten / Einstellungen)
