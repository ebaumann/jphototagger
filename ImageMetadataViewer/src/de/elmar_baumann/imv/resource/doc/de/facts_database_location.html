Wo ist die Datenbank?

Die Datenbank ist im Betriebssystem-Benutzerverzeichnis gespeichert und zwar im Ordner .de.elmar_baumann/ImageMetaDataViewer (Punkt am Anfang). Die Datenbankdateien starten alle mit database.

Dort sind auch die Anwendungseinstellungen gespeichert sowie die letzte Logdatei.

Sie sollten diesen Ordner regelmäßig sichern! Während der Sicherung sollte das Programm 'Bilder' nicht gestartet sein!

Benutzerverzeichnis unter Windows

Unter Windows ist das Benutzerverzeichnis häufig C:\Dokumente und Einstellungen\Benutzername, z.B. C:\Dokumente und Einstellungen\elmar.

Benutzerverzeichnis unter Linux

Unter Linux ist das Benutzerverzeichnis in der Regel /home/benutzername, z.B. /home/elmar.
