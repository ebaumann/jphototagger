Kartenreiter Bearbeiten

Über den Kartenreiter Bearbeiten können Sie die Metadaten eines oder mehrerer Bilder gleichzeitig bearbeiten, sofern Sie Schreibrechte besitzen im Bilder-Ordner und für existierende XMP-Dateien. Ohne Schreibrechte sind die Eingabefelder dunkler und es steht am Ende im Kartenreiter Bearbeiten, dass Bearbeiten nicht möglich ist.

Ihre Daten werden erst übernommen, nachdem Sie diese speicherten! Tipp: Klicken Sie dazu nicht jedesmal auf die Schaltfläche Speichern sondern benutzen das Tastaturkürzel Alt+S. So können Sie schneller eingeben.

Daten eingeben

Mit der Tabulator-Taste gehen Sie von Eingabefeld zu Eingabefeld. Sofern Sie es nicht ausgeschaltet haben, werden Ihnen Vorschläge angeboten, die Sie in dieses Feld irgendwann einmal eingaben.

Zeigen Sie mit der Maus etwas länger auf die Beschriftung eines Felds, erscheint manchmal ein Werkzeugtipp (Tooltip), der etwas genauer beschreibt, was hineingeschrieben werden soll.

Sich wiederholende Daten eingeben

Folgende Daten können mehrfach vorkommen:

  • Stichwörter
  • Weitere Kategorien

Der Bereich für diese Daten sind zweigeteilt: Oben ist eine Liste mit den bisherigen Eingaben, unten ein einzeiliges Feld.

Geben Sie Daten in das einzeilige Feld ein und drücken die Enter-Taste oder klicken mit der Maus auf die Plus-Schaltfläche (+). Dabei wird Ihre Eingabe in die Liste übernommen.

Wollen Sie Einträge aus der Liste löschen, wählen Sie einen oder mehrere an und klicken auf die Minus-Schaltfläche (-).

Die Metadaten eines Bilds bearbeiten

Wählen Sie ein Bild aus, werden seine Metadaten angezeigt. Sie können diese verändern und abspeichern, indem Sie auf die Schaltfläche Speichern klicken (Tastaturkürzel: Alt+S). Sie löschen Daten durch Entfernen aus den Eingabefeldern vor dem Abspeichern.

Die Metadaten mehrerer Bilder bearbeiten

Wählen Sie mehrere Bilder aus, werden keine Metadaten angezeigt. Daten, die Sie eingeben werden in allen Bildern ersetzt, wenn sie nur einmal vorkommen, wie der Name des Fotografen, das Copyright, der Aufnahmeort. Eingaben in Felder mit sich wiederholenden Metadaten - Stichwörter und Kategorien - werden allen ausgewählten Bildern angehängt, sofern sie dort noch nicht existieren. Leer gelassene Felder werden nicht verändert. Die Daten werden in XMP-Filialdateien und die Datenbank gespeichert, nachdem Sie die Schaltfläche Speichern betätigen mit der Maus oder dem Tastaturkürzel Alt+S.

Datenvorlagen

Vorlagen sind Metadaten, die Sie immer wieder benötigen. Einmal abgespeichert werden sie in die Bearbeiten-Felder eingefügt durch Klick auf die Schaltfläche Einfügen (Alt+E) im Kartenreiter Vorlagen. Der Name der aktuellen Vorlage wird angezeigt im Dropdown-Listenfeld.

Für eine Vorlage eignen sich besonders Copyright, Adresse des Fotografen und Stichwörter, die Sie immer wieder benutzen.

Sie können aus bestehenden Metadaten eine Vorlage erzeugen. Entfernen Sie vorher alles, was nicht in die Vorlage soll und klicken anschließend auf die Schaltfläche Als Vorlage speichern (Alt+V).

Die Bedeutung der anderen Schaltflächen unter Vorlagen:

Aktualisieren (Alt+A)
Frischt die aktuelle Vorlage auf mit den Daten der Bearbeiten-Felder
Einfügen (Alt+E)
Fügt die Daten der Vorlage in die Bearbeiten-Felder ein
Löschen (Alt+Ö)
Löscht die aktuelle Vorlage
Umbenennen Alt+M
Benennt die aktuelle Vorlage um

Tipps

Benutzen Sie Vorlagen.

Haben Sie viele Bilder fotografiert, wählen Sie vorher alle aus, die die gleichen Stichwörter, Kategorien etc. erhalten sollen. Dann geben Sie die Stichwörter und Kategorien ein und speichern diese ab. Das wiederholen Sie für andere Stichwörter und Kategorien.
