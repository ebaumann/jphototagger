Löschen

Dieser Befehl löscht die ausgewählten Bilder anhand der aktuellen Anzeige:

  • Links ist ein Ordner ausgewählt: Die Dateien werden im Ordner des Datenträgers gelöscht
  • Links ist eine Bildsammlung ausgewählt: Die Dateien werden aus der Bildsammlung entfernt

Die Entfernen-Taste (Entf oder Del) bewirkt das gleiche.
