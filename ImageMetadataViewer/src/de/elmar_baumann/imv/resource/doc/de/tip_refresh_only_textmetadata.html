Tipp: Nur Text-Metadaten aktualisieren

Haben Sie für ein Bild, das bereits in der Datenbank gespeichert ist

  1. außerhalb des Programms Bilder die Metadaten in der Filialdatei dieses Bilds verändert
  2. das Dateidatum aber auf das alte zurückgesetzt

dann

  1. wählen Sie das Bild aus
  2. klicken die rechte Maustaste
  3. wählen im Menü, das aufklappt, Nur Text-Metadaten aktualisieren

Das geht deutlich schneller, als wenn Sie auch das Vorschaubild neu erzeugen lassen. Sie können ganze Ordner auf diese Weise aktualisieren über den Menübefehl Datei / Ordner einlesen. Dort kreuzen Sie an: Nur Text-Metadaten aktualisieren, keine Vorschaubilder erzeugen (schneller).
