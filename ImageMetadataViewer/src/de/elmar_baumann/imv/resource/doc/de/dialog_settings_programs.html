Einstellungen-Dialog / Bilder öffnen

Bei Doppelklick auf ein Vorschaubild öffnet folgendes Programm das Bild

Wählen Sie ein Programm aus durch Klick auf die Schaltfläche Programm auswählen, das bei Doppelklick auf ein Vorschaubild die Original-Bilddatei öffnen soll. Das kann ein Bildbetrachter sein oder eine Bildbearbeitung.

Weitere Programme zum Öffnen von Bildern

Hier können Sie mehrere Programme hinzufügen, die zum Öffnen von Bildern angeboten werden vom Kontextmenü im Fenster mit den Vorschaubildern. Die Programme erscheinen im Kontextmenü in der gleichen Reihenfolge wie in dieser Liste. Sie können die Reihenfolge ändern, indem Sie ein Programm auswählen und mit einer der Schalftlächen rechts der Liste nach oben oder unten verschieben.
