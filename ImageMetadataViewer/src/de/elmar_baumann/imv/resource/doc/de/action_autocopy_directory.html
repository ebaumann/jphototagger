Bilder von der Kamera-Speicherkarte kopieren

Sie können Bilder von der Kamera-Speicherkarte in ein Verzeichnis kopieren lassen unter zwei Voraussetzungen:

  1. Sie benutzen einen Kartenleser oder die Kamera meldet sich als Laufwerk an
  2. Im Einstellungen-Dialog (Menü Bearbeiten / Einstellungen) haben Sie haben den Ordner mit dem Inhalt der Speicherkarte eingestellt

Alle Bilder der Speicherkarte werden in ein Verzeichnis kopiert mit dem Kopieren-Dialog.

Die Bilder werden nicht automatisch in die Datenbank aufgenommen. Wählen Sie erst das Verzeichnis (erneut) aus, in dem die Dateien kopiert wurden.
