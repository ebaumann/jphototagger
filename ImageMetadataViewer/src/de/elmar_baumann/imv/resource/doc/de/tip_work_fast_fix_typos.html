Tippfehler bei vielen Bildern korrigieren

  1. Wählen Sie im Programmfenster links das Metadatum aus, z.B. ein Stichwort, eine Kategorie, ein Copyright, ... So werden alle Bilder angezeigt, denen dieses Metadatum zugewiesen ist.
  2. Klicken Sie in das Vorschaubildfenster (Strg+0) und wählen alle Bilder aus (Strg+A)
  3. Korrigieren Sie im Kartenreiter Bearbeiten (Strg+E) den Tippfehler
  4. Wählen Sie anschließend ein beliebiges Bild (Ordner etc.) aus, werden die korrigierten Daten gespeichert
