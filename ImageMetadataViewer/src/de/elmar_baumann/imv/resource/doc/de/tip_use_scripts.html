Tipp: Mehr Funktionen hinzufügen

Sie können dem Programm Bilder mehr Funktionen hinzufügen, wenn Sie Skripte (Batch-Dateien) schreiben können und mit Kommandozeilenprogrammen vertraut sind: Sie schreiben Skripte und fügen diese als weitere Programme hinzu. So erscheinen sie im Kontextmenü des Vorschaubildfensters, mit dem sie alle ausgewählten Bilder an Ihre Skripte schicken können.

Beispiel: Über den URL http://imagemagick.org/ erhalten Sie mehrere Kommandozeilenprogramme, mit denen Sie Bilder vielfältig bearbeiten können: Drehen, Schärfen, mit Wasserzeichen versehen, ...

Mit etwas Fantasie ist viel möglich: Hochladen auf einen Webserver, Komprimieren und als E-Mail verschicken, auf eine DVD brennen, eine Webgalerie erzeugen, ...
