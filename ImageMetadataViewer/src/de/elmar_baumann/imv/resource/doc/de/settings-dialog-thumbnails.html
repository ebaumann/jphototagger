Einstellungen-Dialog / Vorschaubilder

In den Bildern enthaltene Vorschaubilder benutzen

Kreuzen Sie hier an, werden die Vorschaubilder geladen, die in den Bildern eingebettet sind. Das geht schneller, als die Vorschaubilder zu berechnen, aber manchmal sind die eingebetteten Vorschaubilder von geringer Qualität.

Maximale Seitenlänge berechneter Vorschaubilder in Pixel

Wollen Sie keine eingebetteten Vorschaubilder benutzen, werden diese so groß berechnet, wie Sie hier angeben. Für RAW-Dateien werden unabhängig von dieser Einstellung immer die eingebetteten Vorschaubilder benutzt, außer Sie lassen die Vorschaubilder mit einem eigenen Programm berechnen, siehe unten!

Anzahl der Vorschaubilder nebeneinander

Im Fenster werden so viele Vorschaubilder nebeneinander platziert, wie Sie hier einstellen.

Eigenes Programm zum Erzeugen der Vorschaubilder

Die Vorschaubilder werden berechnet mit Imagero. Sie können diese mit einem beliebigen anderen Programm erzeugen lassen, sofern dies JPEGs an die Standardausgabe liefern kann. Ein anderes Programm ist sinnvoll, wenn:

  • Sie bei RAW-Dateien nicht die eingebetteten Vorschaubilder haben wollen
  • Ihr Programm besonders hochwertige Vorschaubilder liefert
  • Ihr Programm Vorschaubilder schneller liefert
