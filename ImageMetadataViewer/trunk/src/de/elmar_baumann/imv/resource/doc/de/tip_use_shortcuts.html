Tipp: Tastaturkürzel

Nutzen Sie Tastaturkürzel! Sie erledigen Arbeiten so viel schneller:

  • Alle Kürzel stehen in den Menüs neben dem Befehl. Ausgeführt werden diese durch Drücken der Taste Strg und gleichzeitig eines Buchstabens.
  • Auf Schaltflächen ist der betreffende Buchstabe ist unterstrichen. Das Kürzel wird ausgeführt mit Alt+Buchstabe.
  • Manchmal müssen Sie vorher die Alt-Taste drücken, damit der Unterstrich erscheint.

Beispiel

Sie wollen Stichwörter für ein Bild einfügen: Mit Strg+B springen Sie direkt zur Eingabe und mit Alt+S speichern Sie diese ab.
