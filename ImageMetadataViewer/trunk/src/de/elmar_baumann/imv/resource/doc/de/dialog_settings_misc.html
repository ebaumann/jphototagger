Protokolloptionen (Logdatei)

Der Programmablauf wird protokolliert und das Protokoll in eine Logdatei geschrieben. Diese kann angezeigt werden durch Klick auf eine Schaltfläche am unteren Fensterrand. Sie hat ein grünes Häkchen, wenn alles ok ist und ein rotes Kreuz, falls Fehler auftraten. Bedeutung der Auswahl:

  • WARNUNG: Nur wichtige Fehler und weniger wichtige werden protokolliert; im Normalfall ist das die günstigste Einstellung
  • SCHWERWIEGEND: Nur wichtige Fehler werden protokolliert
  • INFO: Informative Ausgaben, für den Benutzer eines Programms meist unwichtig
  • KONFIG: Ausgaben betreffend die Konfiguration, für den Benutzer eines Programms meist unwichtig
  • FEIN: Detaillierte Ausgaben, insbesondere SQL-Statements, die die Datenbank aktualisieren
  • FEINER: Noch detailliertere Ausgaben, insbesondere SQL-Statements, die die Datenbank abfragen (und nicht modifizieren)
  • AM FEINSTEN: Alles wird protokolliert

Wählen Sie WARNUNG, solange keine Probleme bestehen und zur Fehlerbehebung AM FEINSTEN. Meldungen, die keine Warnungen sind oder keine wichtigen Fehler, werden in die Standardausgabe geschrieben, andernfalls würde die Protokolldatei viele Megabyte groß (erfahrene Computerbenutzer wissen: Die Standardausgabe lässt sich umleiten in eine Datei). Die Standardausgabe sehen Sie im Fenster, über das das Programm Bilder gestartet wurde. Unter Windows ist es minimiert in der Taskleiste.

Ordner der Kamera-Speicherkarte

Hier können Sie einen Ordner auswählen, von dem alle Bilder einer Speicherkarte in einen Ordner kopiert werden.

Wählen Sie den Ordner aus, in dem der Inhalt Ihrer Kamera-Speicherkarte angezeigt wird, wenn Sie Ihre Kamera mit dem Rechner verbinden oder die Karte in einen Kartenleser einführen: Unter Windows einen Laufwerksbuchstaben, unter Linux einen Ordner, beispielsweise /mnt/compactflash. Das Laufwerk / der Ordner werden nach Bildern durchsucht einschließlich aller untergeordneten Ordner.

Webbrowser, z.B. für Google Maps

Hier wählen Sie einen Webbrowser aus, der aufgerufen wird, falls Sie mit dem Internet verbunden werden müssen, beispielsweise zum Anzeigen von Satellitenbildern des Aufnahmeorts bei GPS-Daten im Bild.

Datenbankordner nach Neustart

Sie können die Datenbank in einem anderen Ordner anlegen lassen. Wollen Sie Ihre bisherigen Daten übernehmen, kopieren Sie aus dem aktuellen Datenbankordner, der im Dialog angezeigt wird, bevor Sie es ändern, folgende Dateien nach Beenden von Bilder und vor einem Neustart:

  • Den Ordner thumbnails, dieser enthält die Vorschaubilder
  • Alle Dateien, die mit database starten, z.B. database.data

So können z.B. mehrere - nicht gleichzeitig! - die gleiche Datenbank benutzen und bearbeiten oder Sie wählen einen Ordner aus, der täglich automatisch gesichert wird.

Versteckte Ordner einlesen und anzeigen

Wollen Sie Bilder archivieren, die in versteckten Ordnern gespeichert sind, kreuzen Sie dieses Kästchen an.

Versuchen, nach Programmstart im Kartenreiter Ordner zuletzt ausgewählten Ordner anzuzeigen

Ist diese Option angekreuzt, wird versucht, nach dem Start den Ordner auszuwählen, der beim Beenden von Bilder im Kartenreiter Ordner ausgewählt war. Hat ein Laufwerk viele Ordner, kann dies den Start verzögern. Unter dem Betriebssystem Windows kommt es aus unterschiedlichen Gründen zur Verzögerung, beispielsweise wenn ein Laufwerksbuchstabe mit einem Netzlaufwerk verknüpft war und dieses nicht erreicht werden kann.

Systemausgabenfenster nach Programmstart öffnen

Ist diese Option angekreuzt, wird nach dem Start das Systemausgabefenster geöffnet. Dieses zeigt zusätzlich alle Meldungen an, die nicht in die Protokolldatei geschrieben werden. Sie können das Systemausgabefenster selbst öffnen durch Klicken auf die Schaltfläche Ausgabe am unteren Rand des Programmfensters.
