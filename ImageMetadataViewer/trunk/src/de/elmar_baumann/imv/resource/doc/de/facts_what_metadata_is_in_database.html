Was ist in der Datenbank gespeichert?

In der Datenbank sind Vorschaubilder gespeichert, eine Auswahl an Metadaten und einige weitere Daten:

Aus den XMP-Filialdateien oder in Bilddateien eingebettetem XMP:

  • dc:subject
  • dc:creator
  • dc:rights
  • dc:description
  • Iptc4xmpCore:Location
  • dc:title
  • Iptc4xmpCore:CountryCode
  • photoshop:Headline
  • photoshop:CaptionWriter
  • photoshop:Category
  • photoshop:SupplementalCategories
  • photoshop:City
  • photoshop:State
  • photoshop:Country
  • photoshop:TransmissionReference
  • photoshop:Instructions
  • photoshop:Credit
  • photoshop:Source
  • photoshop:AuthorsPosition

Aus den EXIF-Daten der Bilddatei:

  • DateTimeOriginal
  • FocalLength
  • ISOSpeedRatings
  • Model

Weitere Daten in der Datenbank

  • Automatisch einzulesende Verzeichnisse
  • Bildsammlungen
  • Reguläre Ausdrücke zum Ausschließen von Bilddateien
  • Vorlagen zum Bearbeiten von Metadaten
  • Weitere Programme zum Öffnen von Bildern
  • Gespeicherte Suchen

Nicht in der Datenbank

Alles andere, beispielsweise das Standardprogramm zum Öffnen von Bildern oder die Spalten für die Schnellsuche, ist in der Einstellungen-Datei gespeichert (Settings.properties) im Benutzerverzeichnis.
