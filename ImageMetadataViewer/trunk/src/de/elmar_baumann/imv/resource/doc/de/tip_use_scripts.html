Tipp: Mehr Funktionen hinzufügen

Sie können dem Programm Bilder mehr Funktionen hinzufügen durch Nutzen anderer Programme oder selbst geschriebener Skripte (Batch-Dateien) im Aktionen-Dialog oder im Kontextmenü des Vorschaubildfensters (Einstellungen-Dialog). Beispiele:

  • Drehen von Bildern
  • Ändern einer falsch eingestellten Uhrzeit in den EXIF-Daten
  • Anzeigen einer Diaschau
  • Hochladen auf einen Webserver
  • Eine DVD brennen
  • ...

Empfehlungen

Folgende Programme eignen sich gut dafür:

  • ExifTool: http://www.sno.phy.queensu.ca/~phil/exiftool/ (Bearbeiten von EXIF, IPTC und XMP)
  • ImageMagick: http://imagemagick.org/
  • IrfanView: http://www.irfanview.com/
  • XnView: http://pagesperso-orange.fr/pierre.g/xnview/de_xnview.html

Beispiel-Sktripte

Beispiel-Sktripte sind im Verzeichnis scripts im Installationsordner des Programms Bilder.
