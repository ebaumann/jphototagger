Einstellungen-Dialog > Leistung

Rechenzeit

Prozessor-Leistung für das Berechnen der Vorschaubilder

Sie können einstellen, wieviel Rechenleistung das Programm zum Berechnen der Vorschaubilder und Lesen der Metadaten benutzt. Wollen Sie noch mit anderen Programmen arbeiten, ist die Einstellung Mittel in der Regel geeignet. Sollte der Rechner langsamer reagieren als gewohnt, wählen Sie Niedrig. Das Erzeugen der Vorschaubilder dauert dann länger. Die Einstellung Hoch ist nur geeignet, wenn Sie nicht mehr am Rechner arbeiten müssen, erzeugt die Vorschaubilder aber am schnellsten.

Maximale Wartezeit in Sekunden, bis externe Programme beendet werden

Hier geben Sie ein, nach wievielen Sekunden ein externes Programm beendet wird. Das ist von Belang, falls Sie die Vorschaubilder von einem externen Programm erzeugen lassen und eigene Skripte einsetzen zum Verarbeiten von Bildern. Sollte ein Programm nach dieser Zeit nicht fertig sein, wird es beendet, damit Bilder weiter arbeiten kann.

Start beschleunigen

Auto-Vervollständigen ausschalten

Beim Start werden alle Wörter aus der Datenbank geladen wodurch er sich verzögert. Dafür können Sie schneller eingeben, da bekannte Wörter vorgeschlagen werden.
