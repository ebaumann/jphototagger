Löschen

Dieser Befehl löscht verschieden:

  • Sie haben im Ordner-Kartenreiter einen Ordner ausgewählt und dessen Bilder werden angezeigt: Alle ausgewählten Bilder werden im Ordner des Datenträgers gelöscht
  • Eine Bildsammlung wird angezeigt: Die ausgewählten Bilder werden aus der Bildsammlung entfernt
  • Sie bearbeiten Metadaten und haben in einer Liste ein Metadatum ausgewählt: Das Metadatum wird entfernt

Die Entfernen-Taste (ENTF oder DEL) bewirkt das gleiche.
