EXIF metadata

To display a new tag:

  • Define a new ExifTag
  • Add this tag to ExifMetadataToDisplay#tagsToDisplay
  • Check for presence in de.elmar_baumann.imv.resource.ExifTagIdFieldNameTranslations.properties (all languages)
  • Check for presence in de.elmar_baumann.imv.resource.ExifTagIdTagNameTranslations.properties (all languages)
  • Format tag value in ExifFieldValueFormatter
  • Define display order in ExifIfdEntryDisplayComparator
