How NetBeans automatically inserts the JPhotoTagger license into new created files (using the new file wizard)

The files "project.properties" in all JPhotoTagger project subdirectories "nbproject" are containing a variable definition since 2010-01-05 [NetBeans 6.8]: 

project.license=jphototagger

To add the license automatically when creating a file with the NetBeans Wizard:

  1. Add the file "license-jphototagger.txt" in the root directory of all JPhotoTagger project directories to the "License" templates (Tools > Templates > Licenses; Add-Button). This file contains the license and Freemarker tags. After adding, that file will be copied into Your ".netbeans" user configuration directory (e.g. "$HOME/.netbeans/6.8/config"), subdirectory "Templates/Licenses".
  2. Ensure that in Your Code templates (Tools > Templates > Java, Swing GUI Forms) is this text, beginning at the first line: 
<#-- FreeMarker template (see http://freemarker.org/) -->
<#assign licenseFirst = "/*">
<#assign licensePrefix = " * ">
<#assign licenseLast = " */">
<#include "../Licenses/license-${project.license}.txt">

Tipp: Inserting your name as author

Insert this text into the code templates above the class declaration: 

/**
 *
 *
 * @author  ${user}
 */

The variable user is defined in Tools > Templates > User configuration properties (User.properties), e.g. 

user=Elmar Baumann

All Java code templates are now having this header:

<#-- FreeMarker template (see http://freemarker.org/) -->
<#assign licenseFirst = "/*">
<#assign licensePrefix = " * ">
<#assign licenseLast = " */">
<#include "../Licenses/license-${project.license}.txt">
<#if package?? && package != "">
package ${package};

</#if>
/**
 *
 *
 * @author  ${user}
 */
