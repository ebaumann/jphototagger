Run configurations in the NetBeans IDE

With a Run configuration You can execute a program with different command line options or different Java virtual machine options, using other classes as main class and using different working directories. These are the options whithin JPhotoTagger's source code repository:

  • Language en: Starts the Java VM with the language code en (-Duser.language=en) for testing the English GUI strings
  • Memory Windows User: VM memory settings used whithin windows installations (the installer writes a start batch file with these settings)
  • More Memory: Gives JPhotoTagger more heap memory
  • Output to JPT window: Does not use the command line option -nocapture for testing JPhotoTagger's output window
  • Show Splash Screen: Displays the splash screen

You can switch the configurations either in the toolbar's combo box or via the project settings dialog, category Run.
