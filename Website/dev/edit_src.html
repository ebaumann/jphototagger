Edit source code

Use your favorite IDE or text editor, but existing GUI source-code files created with NetBeans have to be edited with NetBeans. Those files have a sidecar file with the same prefix and the suffix "form" and are located in the same directory, for example, the source code file "ActionsDialog.java" has a sidecar file named "ActionsDialog.form".
