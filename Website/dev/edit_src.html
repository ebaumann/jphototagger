Edit source code

Use Your favorite IDE or text editor, but existing GUI source code files created with NetBeans have to be edited with NetBeans. Those files have a sidecar file with the same prefix and the suffix form and are located in the same directory, e.g. the source code file ActionsDialog.java has a sidecar file named ActionsDialog.form.

Using NetBeans has two main advantages:

  1. In the source repository are existing project files, nothing has to be configured. You can start working immediately after getting the source code.
  2. Kenai Integration including the issue tracker
