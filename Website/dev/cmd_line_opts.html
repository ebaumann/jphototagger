Command line options for Developers

  • Probably the most important one is -nocapture: Do not capture output for displaying it in JPhotoTagger's output window. Else the IDE's output window does not display the system output with the possibility to jump to source code lines on exceptions.
  • -nosplash: Do not draw messages on the splash screen. Important on some Linux systems where this leads to freezing the X window system while dragging.

JPhotoTagger's command line options are defined in the class de.elmar_baumann.jpt.app.AppCommandLineOptions.

Setting the options in the NetBeans IDE

Select in the Projects window the root node of the project, open the context menu and select Properties > Categories > Run. Write the options into the Arguments text field. [Status: NetBeans 6.8]
