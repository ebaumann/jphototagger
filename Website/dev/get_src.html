How to get JPhotoTagger's Source Code

Clone the Mercurial source code repository: 

hg
clone
http://code.google.com/p/jphototagger/source/checkout
jphototagger

This example has 4 lines to fit into this page. On the command line this command has to be on the same line or delimited e.g. whith a backslash within the Bash.

Using the source

After getting the source You can open the projects with the NetBeans IDE: File > Open Project. If NetBeans displays the Reference Problems dialog, You can resolve these through building the projects in that order:

  1. Lib
  2. Plugin
  3. Other used plugins located in the directory Plugins, currently CopyFilenamesToClipboard

To build a project, right click on it's root node in the Projects tab, open the context menu and choose Build.

As next steps You can choose a run configuration and verify the command line options of that configuration.
