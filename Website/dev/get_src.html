How to get JPhotoTagger's Source Code

Clone the Mercurial source code repository:
hg clone https://jphototagger.googlecode.com/hg/ jphototagger

On the command line this command has to be on the same line or delimited e.g. whith a backslash within the Bash.

Using the source code

After getting the source, you can open the projects with the NetBeans IDE: File > Open Project [NetBeans 6.8]. If NetBeans displays the Reference Problems dialog, you can resolve these through building the projects in that order:

  1. JPhotoTagger: Libraray
  2. JPhotoTagger: Plugin: Base
  3. Other used plugins located in the directory Plugins, currently JPhotoTagger: Plugin: Copy Filenames To Clipboard and JPhotoTagger: Plugin: Flickr Upload

To build a project, right click on it's root node in the Projects tab, open the context menu and choose Build.

Next steps:

  • Choose a run configuration and verify the command line options of that configuration.
