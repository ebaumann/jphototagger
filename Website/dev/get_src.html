How to get JPhotoTagger's Source Code

  1. If not already, become a Kenai member
  2. If not done, create on Your local machine a SSH key and add it to Your Kenai profile
  3. Clone the Mercurial source code repository, URI: ssh://your_kenai_username@hg.kenai.com/jphototagger~srcrepo

The easy way

  1. If not done, download and install NetBeans
  2. In NetBeans select the menu item Team > Kenai > Open Kenai Project
  3. Search: JPhotoTagger
  4. Select the found project and apply OK
  5. In the Kenai Panel choose Login to Kenai
  6. Choose get in the node JPhotoTagger > Sources > JPhotoTagger source code repository
