Viewing images: Scripts

In JPhotoTagger you can add image displaying programs through the program window menu: Edit > Settings > 1. Open Images.. Beside selecting a program, you can write and choose a script or batch file. This document contains some script/batch file suggestions.

Let Windows Vista/7 display images

Write into a batch file the following content (thanks to Matthias Vonken):

@rundll32.exe %SystemRoot%\System32\shimgvw.dll, ImageView_Fullscreen %1
