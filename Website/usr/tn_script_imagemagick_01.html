Creating Thumbnails: ImageMagick one liner

  • If not installed, download and install ImageMagick
  • Open: Edit > Settings > Thumbnails
  • Check Create thumbnails with own external program
  • Write into the text field under External image creation application: 
"C:\Program files\ImageMagick\convert" -thumbnail %ix%i -auto-orient "%s" jpg:-

If the ImageMagick is installed elsewhere, fit the path to the tool convert.
