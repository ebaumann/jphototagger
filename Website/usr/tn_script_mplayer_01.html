Creating Thumbnails: MPlayer (Videos), dcraw and ImageMagick in a Bash script (1)

This Bash script also renders a thumbnail of video files.

JPhotoTagger reads Video files only, if their extensions added as User Defined File Types (Menu Window).

  • If not installed, download and install dcraw, ImageMagick and MPlayer
  • Use this File as template, add executable privileges
  • Fix paths
  • Open: Edit > Settings > Thumbnails
  • Check Create thumbnails with own external program
  • Write into the text field under External image creation application:
    "Path_to_Bash_file" "%s" %i
    where "Path_to_Bash_file" is something like "/home/me/scripts/jphototagger_thumbnails.sh" (the folder path where the Bash file is stored is included the Bash script's file name)
