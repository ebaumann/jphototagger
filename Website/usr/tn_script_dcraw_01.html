Creating Thumbnails: dcraw and ImageMagick in a Batch File

Usage for RAW files.

  • If not installed, download and install DCRaw and ImageMagick
  • Create a batch file as listed below
  • Open: Edit > Settings > Thumbnails
  • Check Create thumbnails with own external program
  • Write into the text field under External image creation application:
    "Path_to_Batch_file" "%s" %i
    where "Path_to_Batch_file" is something like "C:\Batch\dcraw.bat" (the folder path where the batch file is stored included the batch file name)

Batch File

Written by Matthias Vonken. You have to fit the paths to dcraw.exe and convert.exe and joining the last lines after "rem" to one line.

@echo off

set IMAGE_FILE=%1
set MAX_LENGTH=%2

rem Creating thumbnail - NEXT LINE HAS TO BE ONE LINE, REMOVE LINE BREAKS (JOINING THE FOLLOWING LINES)
"C:\dcraw\dcraw.exe" -e -c %IMAGE_FILE% 
| "C:\Program Files\ImageMagick-6.6.9-Q16\convert.exe"
 - -thumbnail %MAX_LENGTH%x%MAX_LENGTH% -auto-orient jpg:-
