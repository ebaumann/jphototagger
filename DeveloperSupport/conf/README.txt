This directory contains machine dependend configuration files used by scripts.
They are excluded from the repository and should be fitted to your environment:

googlecode.properties
---------------------
Keys:
username - Google Code username for uploading JPhotoTagger-Setup.exe and -.ZIP
password - Google Code password for uploading JPhotoTagger-Setup.exe -.ZIP

innosetup.properties
---------------------
Keys:
compil32.exe - Path to Inno Setups compiler compil32.exe, e.g. C:\\Program Files\\InnoSetup\\Compil32.exe

website.properties
---------------------
Keys:
server - FTP server for uploading JPhotoTaggers's website
user - Username for FTP server
pass - Password for FTP server
