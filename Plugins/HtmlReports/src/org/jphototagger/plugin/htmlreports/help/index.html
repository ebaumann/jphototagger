Plugin: HTML-Bericht

Das Plugin HTML-Bericht erzeugt aus ausgewählten Vorschaubildern HTML-Dateien mit Tabellen: Jede Tabellenzeile enthält frei definierbare Metadaten eines Bilds, beispielsweise dessen Aufnahmedatum und Stichwörter.

Sie können das Plugin ausführen über das Vorschaubildfenster-Kontextmenü: Plugins > HTML-Bericht.

Einstellungen

Die Berichte passen Sie an über Bearbeiten > Einstellungen > Plugins > HTML-Bericht, alle Optionen sollten "selbsterklärend" sein.

Layout anpassen

Das Layout der HTML-Datei können Sie anpassen durch Verändern der Datei report.css im gleichen Verzeichnis wie die Berichte, hierzu benötigen Sie Wissen über Cascading Style Sheets (CSS). Existiert eine Datei dieses Namens, wird sie beim Erzeugen der Berichte nicht erstellt (nicht überschrieben), Sie können darin Änderungen vornehmen, ohne dass diese beim Schreiben eines neuen Berichts verloren gehen.
