Plugin: Flickr-Upload

Das Plugin Flickr-Upload lädt Bilder hoch nach http://www.flickr.com. Damit Sie das Plugin benutzen können, benötigen Sie einen Flickr Account.

Sie müssen einmalig JPhotoTagger erlauben, Bilder hochzuladen. Dazu wird beim ersten Aufruf dieses Plugins ein Web-Browser geöffnet. Die Erlaubnis können Sie jederzeit widerrufen.

Einstellungen

Erlaubnis für Upload zurücksetzen

Klicken Sie auf die Schaltfläche Zurücksetzen, fragt JPhotoTagger beim nächsten Benutzen dieses Plugins erneut nach Erlaubnis für den Bild-Upload. Das ist erforderlich, falls Sie bei Flickr die Erlaubnis für JPhotoTagger entzogen.

Aus XMP-Filialdatei einfügen

Fügt dem Upload die angekreuzten Metadaten hinzu, falls die Bilddatei eine XMP-Filialdatei hat und die Filialdatei die angekreuzten Daten enthält. Diese stehen auf Flickr zusammen mit dem Bild:

  • Stichwörter: Fügt die Stichwörter als Tags hinzu
  • Bildtitel: Erscheint als (große) Bildüberschrift
  • Bildbeschreibung: Steht unter dem Bild
